🔗 https://portfolio-website-ej435w4ag-tosifansaris-projects.vercel.app/

## 📌 Features

- Responsive Design
- Modern UI
- Smooth Navigation
- Projects Showcase
- Skills Section
- About Me
- Contact Section

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript
- HTML5
- CSS3

## 📂 Installation

Clone the repository

```bash
git clone https://github.com/tosifansari/portfolio-website.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
